
Headings

Use hash symbols (#) to create headings. The number of hashes determines the heading level, from H1 (largest) to H6 (smallest). 
# Heading 1
## Heading 2
### Heading 3

## Text Styling

Enclose text within special characters for emphasis. 
Bold: **bold text** or __bold text__
Italic: *italic text* or _italic text_
Strikethrough: ~~strikethrough text~~
Bold and italic: ***bold and italic text*** 

## Lists

Create ordered and unordered lists. 

Unordered lists: Use asterisks (*), hyphens (-), or plus signs (+).
* First item
* Second item
* Nested item (indent by two or four spaces)

Ordered lists: Use numbers followed by a period.
1. First item
2. Second item
3. Third item

Task lists: * [ ] for unchecked and * [x] for checked items (GitHub Flavored Markdown).
- [x] A completed task
- [ ] An uncompleted task 

## Links and Images

Inline links: [Anchor Text](URL) (e.g., [Markdown Guide](https://www.markdownguide.org/)).

Images: ![Alt text](image_url). 

## Code
Inline code: Enclose code snippets in single backticks (`code`).

Code blocks: Enclose multiple lines of code in triple backticks (```). You can add an optional language identifier for syntax highlighting.

```javascript
var s = "JavaScript syntax highlighting";
alert(s);
```
 
## Other Elements

Blockquotes: Start a line with an angle bracket (>) to quote text.
> This is a blockquote.

#### Horizontal rule: 
#### Use three or more hyphens, asterisks, or underscores on a line by themselves.
---

### Tables: 

Use pipes (|) to separate columns and hyphens (-) to create the header row.
| Header 1 | Header 2 |
| --- | --- |
| Cell 1 | Cell 2 |

### Footnotes: 
Here is a sentence with a footnote.[^1] (define the footnote elsewhere: [^1]: This is the footnote.). 

### Format/Emphasis

For emphasis, use asterisks or underscores:

Italics: *italic* or _italic_

Bold: **bold** or __bold__

Bold and Italic: ***bold and italic***

underline: <u> This sentence will be underlined. </u>
